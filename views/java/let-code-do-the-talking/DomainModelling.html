Introduction

Lets design a Parking Lot. Consider that there is a multi-level parking lot of a building which houses multiple companies. The parkings are available for Cars and Bikes. Lets understand the requirement and design it simultaneously as an Agile Development Methodology using stories.

Parking Lot has multiple levels. Some levels are reserved for Bike parkings and some for Car parking.
